Fortress (software)

From Wikipedia, the free encyclopedia

For other uses, see Fortress (disambiguation).

Fortress is a commercial, proprietary Application Lifecycle Management tool by SourceGear LLC which markets its product as an integrated version control and issue tracking system.

Fortress replaces SourceGear's previous Dragnet bug tracking tool, and integrates it with the version control features of SourceGear Vault.

The tool is built on top of Microsoft .NET, and offers Visual Studio and Eclipse integration, a Windows stand-alone client, and Windows, Linux and Mac OS command-line clients.
